O que é um Jupyter Notebook?¶

O jupyter notebook e JupyterLab so uma aplicaes tanto para desenvolver e executar cdigos quanto elementos de texto, figuras, links, vdeos, equaes, etc. Como podemos misturar texto e cdigo em um mesmo lugar, estes documentos ficam ideais para gerar e descrever anlises onde tanto a formulao da ideia, a visualizao do resultado e a descrio do resultado pode ser feita num mesmo lugar.

  • Documentao do IPython API
  • Site oficial do Jupyter

Células¶

Isso uma clula de texto. Clulas de texto aceitam marcao conhecida como Markdown. Em uma clula de cdigo ns podemos usar o Python como usamos em um script qualquer:

In [12]:
# isso  uma clula de cdigo

for count in range(0,5):
    print(count)
print('end')

0
1
2
3
4
end

Cada clula um ambiente isolado que pode ser rodado separadamente. Voc pode rodar cada clula individualmente e na sequncia que desejar, mas as variveis que ela usa precisam estar referenciadas antes do seu uso.

OBS: caso haja dependencia entre celulas, garanta que todas as celulas foram executadas (run all cells)

Teclas de atalho¶

  • Tab = code completion or indent
  • Shift + Tab = tooltip
  • Ctrl + Enter = run cell
  • Y = code
  • M = markdown

Fórmulas matemáticas¶

Tambm possvel usar frmulas junto com o texto. Isso muito til quando se tem uma explicao matemtica necessria para entender o cdigo. A escrita feita da mesma forma que no LaTeX:

$$c = \sqrt{a^2 + b^2}$$
